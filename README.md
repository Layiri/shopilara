## ShopiLara. Get Products and collections from shopify 

This application allows to download on its site based on Laravel, its products and collections from shopify.
## Requirements
 PHP 7.4, Laravel 8, MySql 5.7.28

## How to use?
Create a shop on shopify.
Link the shopify shop, with the application created on the partners.shopify site.
Next, remove the desired API, API_secret, and Scopes.
